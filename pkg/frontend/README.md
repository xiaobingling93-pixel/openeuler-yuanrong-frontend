# Faas Frontend

See the Frontend structure as below, 

```text
+------+     +------------+     +----------------+     +--------------+     +----------------+
|      | --> | Invocation | --> |                | --> | InstancePool | --> | FaaS Scheduler |
|      |     +------------+     | FunctionInvoke |     +--------------+     +----------------+
|      |     +------------+     |                |     +--------------+
|      | --> |  Trigger   | --> |                | --> |  Go Runtime  |
| HTTP |     +------------+     +----------------+     +--------------+
|      |     +------------+     +----------------+
|      | --> |   Alias    | --> |  Alias Route   |
|      |     +------------+     +----------------+
|      |     +------------+     +----------------+
|      | --> |   Worker   | --> |  Worker Route  |
+------+     +------------+     +----------------+
```
